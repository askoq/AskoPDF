## How to run viewer

Make sure you have [Git](https://git-scm.com/) and [Flutter](https://docs.flutter.dev/get-started) installed.

To run the viewer engine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/askoq/askopdf.git
   cd askopdf

2. Restore project files (Important): Since this is a lightweight repo, generate the platform folders:
   ```bash
   flutter create .

2. **Install dependencies:**
   ```bash
   flutter pub get

3. **Run:**
   ```bash
   flutter run -d windows
